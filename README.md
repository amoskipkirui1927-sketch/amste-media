# Amste Print Media - Professional Printing Services Platform

A full-stack web application for managing printing services with real-time communication between customers and admin.

## Features

- Customer registration and login
- File upload for printing (images and documents)
- Real-time order tracking
- Admin dashboard with order management
- Live chat between customers and admin
- Profit analytics with charts
- User management
- Pricing configuration
- Sales reports

## Technology Stack

- **Backend**: Node.js, Express.js, Socket.IO
- **Frontend**: HTML5, CSS3, JavaScript
- **Database**: JSON file-based storage
- **Real-time**: WebSockets (Socket.IO)
